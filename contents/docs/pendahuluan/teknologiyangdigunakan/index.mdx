---
title: Teknologi Yang Digunakan
description: Teknologi yang akan saya gunakan selama pengembangan projek ini
---

Pengembangan aplikasi ini akan menggunakan teknologi berikut:

- Frontend: HTML, CSS (dengan framework Bootstrap), dan JavaScript.
- Backend: PHP sebagai teknologi pemrosesan server.
- Basis Data: MySQL sebagai sistem manajemen basis data relasional.
- Deployment: Vercel atau GitHub untuk proses pengujian dan distribusi aplikasi. 
Hosting berbayar dapat dipertimbangkan sebagai opsi tambahan untuk implementasi penuh.
