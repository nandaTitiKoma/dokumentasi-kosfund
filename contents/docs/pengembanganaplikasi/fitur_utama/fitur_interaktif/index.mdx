## Fitur Tracking Menggunakan Chart

### 1. Line Chart (Diagram Garis)
- **Fungsi:**  
  Menampilkan tren pemasukan dan pengeluaran pengguna berdasarkan waktu. Membantu pengguna memahami pola keuangan mereka.  

- **Implementasi:**  
  1. Data pemasukan dan pengeluaran dikumpulkan dari database.
  2. Data diproses untuk menghasilkan total berdasarkan waktu (misalnya harian atau bulanan).
  3. Data divisualisasikan menggunakan library seperti Chart.js dalam bentuk diagram garis.

- **Hasil:**  
  Diagram garis memberikan gambaran tentang:
  - Apakah pengeluaran melebihi pemasukan.
  - Tren keuangan selama periode tertentu.

### 2. Doughnut Chart (Diagram Donat)
- **Fungsi:**  
  Menampilkan distribusi budget dalam berbagai kategori, seperti kebutuhan pokok, hiburan, dan tabungan.  

- **Implementasi:**  
  1. Data budget dikumpulkan dari database.
  2. Persentase alokasi dihitung untuk setiap kategori.
  3. Data divisualisasikan dalam bentuk diagram donat menggunakan library seperti Chart.js.

- **Hasil:**  
  Diagram donat membantu pengguna untuk:
  - Melihat persentase pengeluaran di setiap kategori.
  - Memastikan alokasi budget sesuai rencana.
