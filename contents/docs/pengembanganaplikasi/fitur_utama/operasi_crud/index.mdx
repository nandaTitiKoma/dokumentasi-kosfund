## Operasi CRUD dalam Pencatatan Keuangan

### 1. Create (Membuat Catatan Keuangan)
Pengguna dapat melakukan:
- **Pencatatan Pemasukan:**  
  Pengguna memasukkan jumlah pemasukan, tanggal, dan sumber pemasukan melalui form. Data ini disimpan di database.  
- **Pembuatan Budget:**  
  Pengguna membagi pemasukan ke dalam kategori budget (misalnya kebutuhan sehari-hari, hiburan, tabungan).  
- **Pencatatan Pengeluaran:**  
  Pengguna mencatat pengeluaran dengan memilih kategori budget, jumlah, dan deskripsi transaksi.  

### 2. Read (Melihat Catatan Keuangan)
- Semua data keuangan ditampilkan dalam tabel di dashboard:
  - Pemasukan terurut berdasarkan tanggal.
  - Alokasi budget yang sudah dibuat.
  - Pengeluaran beserta kategorinya.
- Data divisualisasikan menggunakan diagram untuk membantu analisis keuangan.

### 3. Update (Memperbarui Catatan Keuangan)
- Pengguna dapat memperbarui data dengan langkah berikut:
  1. Pilih data pemasukan, budget, atau pengeluaran dari tabel.
  2. Edit data di form yang tersedia.
  3. Simpan perubahan, data diperbarui di tabel dan diagram.

### 4. Delete (Menghapus Catatan Keuangan)
- Pengguna dapat menghapus catatan pemasukan, budget, atau pengeluaran.
- Sistem meminta konfirmasi sebelum data dihapus dari database.
