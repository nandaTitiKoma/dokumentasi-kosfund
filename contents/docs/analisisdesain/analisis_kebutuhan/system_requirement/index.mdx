## Client-Side Requirements

Kebutuhan sistem yang harus dipenuhi oleh perangkat pengguna untuk mengakses aplikasi:

#### Perangkat.

- Komputer, laptop, atau perangkat seluler dengan spesifikasi minimum: RAM: 2 GB atau lebih, Prosesor: Dual-Core atau lebih tinggi.
- Sistem operasi: Windows 10, macOS Mojave, Android 8.0, atau iOS 13 (atau versi lebih baru).


#### Browser

Browser modern seperti:
- Google Chrome (versi 90 ke atas).
- Mozilla Firefox (versi 88 ke atas).
- Safari (versi 14 ke atas).
- Microsoft Edge (versi Chromium).


#### Koneksi Internet
Stabil dengan kecepatan minimum 10 Mbps untuk pengalaman penggunaan yang lancar.



## Server-Side Requirements

Spesifikasi sistem yang diperlukan untuk mendukung operasional aplikasi di sisi server:

#### Perangkat Keras(Hardware).

- CPU: Prosesor minimal Quad-Core.
- RAM: 4 GB atau lebih.
- Penyimpanan: SSD dengan kapasitas minimum 20 GB untuk aplikasi dan basis data.
- Koneksi Internet: Kecepatan upload dan download minimum 10 Mbps untuk mendukung akses pengguna simultan.


#### Perangkat Lunak (Software)

- Sistem operasi server: Ubuntu 20.04 atau CentOS 8.
- Server aplikasi: Apache 2.4 atau Nginx.
- Bahasa pemrograman backend: PHP (versi 7.4 ke atas).
- Basis data: MySQL (versi 8.0 ke atas).



## Development Environment Requirements   

Persyaratan perangkat dan perangkat lunak yang diperlukan oleh pengembang untuk membangun aplikasi:

#### IDE/Editor Kode

- Visual Studio Code.


#### Perangkat Lunak Pendukung

- Node.js untuk pengelolaan dependensi frontend.
- Git untuk kontrol versi.
- XAMPP untuk menjalankan server lokal



## Development Environment Requirements   

Persyaratan perangkat dan perangkat lunak yang diperlukan oleh pengembang untuk membangun aplikasi:

#### Platform Hosting

- Vercel untuk deployment frontend.
- Hosting berbayar (opsional) untuk backend dengan dukungan PHP dan MySQL.
