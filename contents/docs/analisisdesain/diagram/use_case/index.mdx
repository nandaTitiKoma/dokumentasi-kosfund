Aplikasi yang akan saya kembangkan adalah sebuah aplikasi berbasis web untuk pencatatan keuangan.
Aplikasi ini dirancang untuk membantu pengguna mencatat keuangan mereka, meliputi pemasukan, pengeluaran, 
dan pengaturan anggaran dalam jangka waktu tertentu.
Salah satu rencana pengembangan jangka panjang aplikasi ini adalah fitur budgeting otomatis berbasis
kecerdasan buatan (AI) untuk mempermudah pengguna dalam membuat perencanaan keuangan yang cepat, mudah, dan akurat.

Namun, pada fase pengembangan ini, fokus utama adalah menyediakan fitur inti, yaitu:

- Membuat catatan keuangan.
- Menghapus catatan keuangan.
- Mengedit catatan keuangan.

Fitur ini diharapkan menjadi fondasi utama sebelum mengintegrasikan fungsi yang lebih kompleks di masa depan.
